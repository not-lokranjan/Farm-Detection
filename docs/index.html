DetectField

Raspberry Pi computer vision for farm surveillance. YOLO watches the camera feed for humans and wildlife, then the local dashboard opens the live feed and records detection events.

Open Pi Dashboard View Code

Detection

Uses yolov8m-oiv7.pt to identify people and animals including birds, cats, dogs, cattle, goats, horses, deer, foxes, rabbits, monkeys, snakes, pigs, sheep, chickens, ducks, lions, tigers, and cheetahs.

Live Dashboard

The Pi runs Flask on port 8080 with surveillance controls, MJPEG live feed, detection notifications, CPU, memory, temperature, voltage, throttle flags, and uptime.
