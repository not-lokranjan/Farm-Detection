About DetectField

DetectField runs animal and human detection directly on a Raspberry Pi using the attached camera. The dashboard exposes surveillance controls, live camera access, device health, and event alerts over the local network. Detection happens on the Pi, so any phone, laptop, or tablet on the same network can operate the system through this page.
